Craft-ContactFromEntries
========================

Saves emails sent with P&amp;Ts Contact Form plugin to a channel.


The following entry fields will recieve the corresponding email fields:

* title => subject
* body => message
* fromEmail => fromName
* fromEmail => fromEmail
